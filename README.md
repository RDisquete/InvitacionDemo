# 📸 Wedding Album — R&M 2026

Aplicación web desarrollada para un evento real, pensada para que los invitados puedan acceder fácilmente a la información de la boda y compartir fotos en tiempo real desde el móvil.

El objetivo del proyecto es que cualquier persona, sin conocimientos técnicos, pueda entrar, ver el contenido y subir recuerdos de forma rápida y sencilla.

---

## ⚠️ Demo

Esta es una demo funcional del sistema. Algunas imágenes o contenidos pueden estar limitados por privacidad del evento real.

---

## 📲 ¿Para qué sirve el QR?

El QR que reciben los invitados da acceso directo a la aplicación desde el móvil durante el evento.

No requiere registro ni configuración: simplemente escanear y entrar.

---

## 💡 Qué puedes hacer en la aplicación

- Ver el álbum de fotos del evento  
- Subir imágenes directamente desde el móvil  
- Acceder a la información del evento  
- Navegar por las secciones de la boda  

---

## 📸 Sistema de álbum compartido

Los invitados pueden subir fotos en tiempo real durante el evento.

Las imágenes se almacenan en la nube y se muestran automáticamente en la galería para el resto de usuarios.

---

## 📡 Acceso incluso sin conexión estable

El menú del evento está diseñado para seguir funcionando aunque la conexión sea inestable.

Se utiliza caché para mantener accesibles las secciones principales de la web durante el evento.

---

## ⚡ Flujo de uso

Escaneas el QR → accedes a la web → ves la información del evento → subes fotos → compartes recuerdos

---

## 🧠 Qué incluye el sistema

- Sistema de acceso privado  
- Álbum de imágenes compartido  
- Subida de fotos desde móvil  
- Optimización de imágenes en cliente  
- Diseño centrado en experiencia de usuario real  
- Flujo simplificado sin fricción  

---

## 🛠️ Stack tecnológico

- React 18 + TypeScript  
- Tailwind CSS  
- Supabase (base de datos + storage)  
- Framer Motion  
- React Router  
- Formspree
